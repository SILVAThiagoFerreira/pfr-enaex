Plano de Fogo Realizado

Interface web e pipeline Python para gerar o output operacional.

Este projeto possui uma interface Flask para anexar inputs, validar os arquivos e baixar o plano gerado em Excel.

GitHub Pages hospeda apenas arquivos estáticos e não executa backend Python. Para usar upload, processamento e download do XLSX, rode o projeto localmente:

python main.py --web --config config.yaml

Depois de iniciar, acesse http://127.0.0.1:5000. O código-fonte, os testes e a configuração de publicação estão neste repositório.
